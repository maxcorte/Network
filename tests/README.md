## How to launch the tests?

The tests will run automatically by typig this following line in the terminal:

````
./tests/test_performances.sh
````

In the case this didn't work, you first need to allow your computer to execute this file. To do so, go in 
your terminal and type: 

````
chmod +x tests/test_performances.sh
````

and then: 

`````
./tests/test_performances.sh
`````

After this, the program should works instantly.
